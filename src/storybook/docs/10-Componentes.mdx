## Padrao recomendado para novos componentes

1. Criar `Componente.tsx`
2. Criar `Componente.types.ts`
3. Exportar no `index.ts` da pasta
4. Documentar neste catalogo e, quando aplicavel, adicionar `*.stories.tsx`
