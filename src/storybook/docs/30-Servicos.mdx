# Servicos e contratos

## PostService

Arquivo: `src/services/post-service/post-service.ts`

Metodos publicos:

- `getAll({ limit = 10, currentPage = 1 })`
- `getBySlug(slug: string)` — usa `getByUID` no Prismic
- `getAllSlugs()`

## GithubService

Arquivo: `src/services/github-service/github-service.ts`

Metodos publicos:

- `getProfile()`
- `getRepos()`

Usado pela pagina `/about` para exibir estatisticas e repositorios em destaque.

## Comportamento importante de resiliencia

- Cache de leitura em `getAllPosts` usando `react cache`
- Retry unico para falhas de rede em chamadas ao Prismic
- Em producao, falhas de rede com configuracao Prismic valida interrompem o carregamento (fail-fast)
- Em desenvolvimento, fallback para lista vazia com warning unico no console
- GithubService valida shape da resposta e usa fallback seguro

## Transformacoes no PostService

- Conversao `PrismicDocument` -> `BlogPost`
- Normalizacao de tags com fallback para `document.tags`
- Calculo de tempo de leitura com base em 200 palavras/minuto
- Paginacao local via utilitario `paginationPosts`

## Exemplo de uso

```ts
import { GithubService, PostService } from '@/services'

const { posts, currentPage, numbPages } = await PostService.getAll({
  limit: 10,
  currentPage: 1,
})

const post = await PostService.getBySlug('meu-slug')
const profile = await GithubService.getProfile()
```
