## Introdução

Nesse post vamos falar sobre algumas dicas de CSS que podem ser úteis para você.

## Animações

A primeira dica é sobre animações, você pode usar o CSS para criar animações de forma simples e rápida.

```css
@keyframes example {
  0% {
    background-color: red;
  }
  25% {
    background-color: yellow;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

## Gradientes

A segunda dica é sobre gradiente, você pode usar o CSS para criar gradiente de forma simples e rápida.

```css
background: linear-gradient(to right, #ff0000, #0000ff);
```

## Flexbox

A terceira dica é sobre flexbox, você pode usar o CSS para criar layouts de forma simples e rápida.

```css
.container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}
```
